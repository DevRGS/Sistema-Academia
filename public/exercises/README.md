# GIFs de Exercícios

Esta pasta contém os GIFs de demonstração dos exercícios.

## Como adicionar GIFs

1. Baixe ou crie GIFs de demonstração para cada exercício
2. Nomeie o arquivo conforme o mapeamento em `src/utils/exerciseGifs.ts`
3. Coloque o arquivo nesta pasta (`public/exercises/`)

## Recomendações

- **Tamanho**: Mantenha os GIFs com tamanho razoável (máximo 2-3MB)
- **Duração**: GIFs de 3-5 segundos são ideais
- **Qualidade**: Use boa qualidade visual, mas otimize o tamanho do arquivo
- **Formato**: Use formato GIF animado ou considere usar vídeos MP4 (seria necessário atualizar o código)

## Fallback

Se um GIF não estiver disponível, o sistema mostrará uma mensagem amigável informando que a demonstração será adicionada em breve.
